# OS Research FullCalendar

## How to run locally

To start the development server, run:

```bash
npm run dev
```

*Note: If you get a "running scripts is disabled" error in PowerShell, use:*
```powershell
cmd /c npm run dev
# OR
npm.cmd run dev
```

Then open your browser and navigate to:
[http://localhost:5173/](http://localhost:5173/)
